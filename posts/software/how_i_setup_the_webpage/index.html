This website is my personal portfolio and blog. It uses Hugo framework, Emacs, Org-mode, Org-roam and OX-hugo for managing website.

Brief overview of myself:

I am Flow Assurance Team lead in ExxonMobil India. I have worked on various greenfiled and brownfiled projects as FA lead or as independent contributor.

What can you expect here?

This website is a location for me to discuss various topics I learned. The blog is a place for me to dump all my learning and ideas. I would Like to use this as a platform for me to try explain everything I know or learned so that I can reinforce my knowleadge. I know its a selfish thought, but I expect it to useful to anyone who has any interest in Chemical Engineering and Flow Assurance Engineering.

How I setup the webnsite?

I am using HUGO framework to power the site. Its a free opne-source site generator which can be used if you are savy enough to put some effort to learn yaml or toml. It can be used to generate various static sites that are very fast and beautiful.

  • am working on linking up my org-roam database to the blog/notes section of the site so that I can use it and orgmode along with the ox-hugo extension to build the site.
